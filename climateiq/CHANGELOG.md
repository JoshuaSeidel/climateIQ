# Changelog

## 0.1.0 - Initial Release

### Added

- Home Assistant add-on with ingress support
- Embedded PostgreSQL and Redis for zero-configuration deployment
- Nginx ingress proxy with WebSocket support
- MQTT auto-discovery from Home Assistant Supervisor API
- Multi-zone HVAC management dashboard
- Real-time sensor monitoring via WebSocket
- AI-powered chat interface with multi-provider LLM support
  - Anthropic Claude
  - OpenAI GPT
  - Google Gemini
  - xAI Grok
  - Ollama (local inference)
- Smart scheduling with time-based temperature profiles
- Weather integration for proactive climate adjustments
- Energy usage analytics
- Support for amd64, aarch64, and armv7 architectures
